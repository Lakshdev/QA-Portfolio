# SauceDemo E-Commerce Automation

## Overview

This project automates the complete purchase process of the SauceDemo website using Selenium WebDriver.

## Technologies

- Java
- Selenium WebDriver
- TestNG
- Maven
- Eclipse

## Framework Features

- Page Object Model (POM)
- BaseTest
- Assertions
- Screenshot Utility
- End-to-End Automation

## Test Flow

- Login
- Add Product
- Cart
- Checkout
- Finish Order
- Verify Success
