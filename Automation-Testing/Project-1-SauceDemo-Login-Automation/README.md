# 🚀 SauceDemo Login Automation

## Project Overview

This project automates the login functionality of the SauceDemo website using Selenium WebDriver and Java.

It was developed as part of my QA Automation learning journey.

---

## Website

https://www.saucedemo.com

---

## Technologies Used

- Java 21
- Selenium WebDriver
- Maven
- TestNG
- Eclipse IDE

---

## Features Automated

✅ Launch Chrome Browser

✅ Open SauceDemo Website

✅ Enter Username

✅ Enter Password

✅ Click Login

✅ Verify Login Success

✅ Verify Products Page

✅ Close Browser

---

## Selenium Concepts Used

- WebDriver
- ChromeDriver
- Locators
  - ID
  - Name
  - CSS Selector
  - XPath
- Assertions
- TestNG

---

## Project Structure

src/test/java

- LoginTest
- LoginAssertionTest
- LocatorPractice
- LoginTestNG

---

## Expected Result

The automation:

1. Opens Chrome
2. Logs into SauceDemo
3. Verifies successful login
4. Displays PASS
5. Closes browser
